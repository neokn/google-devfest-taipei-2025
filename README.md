# google-devfest-taipei-2025

想快速打造 AI 應用？Firebase Genkit 讓開發 LLM 應用變得優雅又高效。

本 workshop 用 Genkit + Gemini 打造 AI ChatBot，展示 GenKit 如何解決實際痛點：dotPrompt 讓 Prompt 管理不再是技術債、Developer UI 讓調教過程視覺化、Flow 設計讓複雜邏輯模組化、內建 Trace 讓除錯超簡單。

你將學到：
✨ Genkit 的核心概念與開發體驗
✨ dotPrompt 的版本控制最佳實踐
✨ 如何利用 Genkit 加速你的 model 選擇以及 prompt tuning

主要語言使用 TypeScript
參加活動前需確保電腦中已經有 nodejs lts (22 or 24) 版本
以及建議使用 pnpm 作為 package management

此 Workshop 將以 Discord Bot 為例
請事先確認已經有 Discord 帳號
